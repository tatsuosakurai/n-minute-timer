# n-minute-timer

Simple static timer app built with HTML and JavaScript.

## Files

- `index.html`
- `script.js`

## GitHub Pages

This repository is ready to publish with GitHub Pages from the repository root.

Live site: https://tatsuosakurai.github.io/n-minute-timer/

## URL Parameters

Open the app with `/10` to prefill the timer with 10 minutes.
The query forms `?m=10` and `?minutes=10` still work too.
